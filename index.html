Sovnet NFTs are collectible NFTs on Bitcoin Cash using the new CashTokens technology! Sovnet combines hand-drawn ninja art with an ambitious roadmap and an innovative minting setup. Further, there is rich & unique ninja lore to explore!

Sovnet mint uses the WalletConnect v2 standard to connect your wallet with our web3 minting app. Currently this means you can use the Cashonize webwallet, the Zapit wallet or the Paytaca wallet. The only other requirement is to have enough Bitcoin Cash in your wallet to mint your desired number of Sovnet NFTs!

Sovnet NFTs can be traded on Tapswap, a non-custodial marketplace for CashTokens! Tapswap also uses WalletConnect so you can use all the same wallets as for the Sovnet mint!

The goal of Sovnet is to do something fun, cool & unique in the CashTokens ecosystem and to create an enthusiastic community around it! Further, the goal is to contribute to the nascent NFT space on BCH by building open-source tooling to simplify creating NFT projects on Bitcoin Cash.

All of it, so take a look at our GitHub page!

We are committed to simplifying the process of creating a CashTokens NFT collection so we open-sourced every tool we developed, from the generation of the randomized artwork, to the setup of the minting contract, as well as the code of the simple API server and the code for the minting page.

The Sovnet minting contract is multi-threaded allowing for multiple mints at the same time without running into issues! Additionally, Sovnet uses a novel metadata strategy so your Sovnet NFTs shows in your wallet correctly right away.
